# Selenium Automation by Junit

## Project Description:
Learning several webelement handling using selenium and junit by automation a localhost practice site

## Technology Used:
- Selenium
- Junit
- SimpleJSON
- Gradle

## How to run?
- Run the localhost project
- Clone this project: `git clone https://github.com/salmansrabon/test-practice-site-automation-junit-psdet-b18 `
- Hit this command: `gradle clean test`
- Will generate a report
